# AI 人才複製系統課

## 專案目的

建立一個可持續迭代的課程銷售網站與知識庫。

此專案不只是 Landing Page，而是未來 AI 系統課程、模板、陪跑班、企業服務的核心資產。

---

## 專案目標

第一階段

* 完成一頁式銷售頁
* 提供課程報名入口
* 收集潛在客戶名單

第二階段

* 增加課程介紹頁
* 增加案例頁
* 增加 FAQ

第三階段

* 建立會員系統
* 建立學員專區
* 建立企業服務頁

---

## 核心課程

課程名稱：

AI 人才複製系統課

核心主張：

把公司裡最難複製的人才經驗，
整理成 AI 可以執行的系統。
